## Motivador

O Instituto de Economia, Finanças e Adminitração da Aeronáutica **(IEFA)** tem como visão disponibilizar uma suite de soluções para atender as necessidades dos militares subordinados a Secretaria de Economia, Finanças e Administração da Aeronáutica **(SEFA)** e em especial o **Oficiais Intendentes**.

## Estrutura

O Projeto utiliza Pnpm Workspaces para manter diversas aplicações em um Mono Repositório para facilitar a manutenção.
