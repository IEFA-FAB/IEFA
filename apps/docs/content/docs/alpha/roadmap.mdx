## Faseamento

### Fase 1 — Módulo ACI (Etapas 1–8)

Prioridade máxima. Valida o núcleo jurídico antes de escalar.

- **Piloto:** uma OM selecionada
- **Critério de sucesso:** módulo ACI funcional com aprovação da equipe de controle interno
- **Dependência crítica:** mapeamento do schema JSON (Etapa 4) é pré-requisito para Etapas 5–9

### Fase 2 — Módulo Requisitante (Etapas 9–12)

Após validação da Fase 1.

- **Expansão:** abre para requisitantes da OM piloto
- **Critério de sucesso:** requisitante consegue estruturar demanda e gerar documento conforme sem intervenção manual do ACI

### Fase 3 — Copiloto Integrado (Etapa 13)

Após consolidação das Fases 1 e 2.

- **Escala:** abertura progressiva para demais OMs
- **Critério de sucesso:** agente conduz fluxo completo com intervenção humana apenas na aprovação final
